2、解题方法
2-1、双指针法
用于解决在【一维数组】变换，定义两个指针，通常用作【比较两个值】或者【比较某个target值】，当满足条件时进行【交换or计算or比较等操作，接着移动指针】，双指针又可以分为以下两类：

2-1-1、左右指针
数组的0下标定义为左指针left，
数组的末尾下标定义为右指针right，
通过分别从左右向中间移动，避免两次循环，保持时间复杂度为O(N)；
解题的【伪代码】如下：
while (左指针 < 右指针) {
    if (条件成立) {
        计算
        left++ 或者 right--;
    }
}
